ESP32PP Web Installer

Flash your ESP32PP Portapack add-on board.

Flash Firmware

Warning: This process will erase all data and settings on the ESP32PP!

After flashing, you may need to wait a few minutes for the device to show up on Wi-Fi or the PortaPack. Please be patient.

First-Time Setup

Important: On the first run, you MUST open the Web UI to set up your pin configuration. You can choose a template or enter pins manually. If you make a mistake, just come back here and re-flash.

Default Settings

  • Wi-Fi AP: ESP32PP
  • Password: 12345678
  • AP IP Address: 192.168.4.1
